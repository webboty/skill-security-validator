# Skill Security Validator

A security scanner for AI agent skills and scripts. Detects malware, trojans, credential theft, and 200+ malicious code patterns.

## Install

```bash
npx skills add webboty/skill-security-validator
```

## Usage

```bash
python3 ~/.agents/skills/skill-security-validator/scripts/validate_security.py /path/to/skill
```

See [skills/skill-security-validator/README.md](skills/skill-security-validator/README.md) for full documentation.
